# Razeen Fatima Portfolio

A terminal-inspired personal portfolio designed to showcase my projects, skills, and journey as a BS Information Technology student at Bahria University.

Built with a minimalist command-line aesthetic, the portfolio highlights my work in front-end development, Java applications, and C++ programming while maintaining a clean and interactive user experience.

## Features

- Terminal-style interface
- Responsive design
- Interactive project showcase
- Skills and technology stack section
- Resume download
- Contact links and professional profiles

## Built With

- HTML5
- CSS3
- JavaScript
- GSAP

Inspired by developer terminals, clean UI design, and modern portfolio experiences.
